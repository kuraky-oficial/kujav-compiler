# KUJAV Compiler

Kujav es un compilador experimental para un lenguaje fuertemente tipado que genera bytecode JVM (`.class`) y empaqueta `.jar`.

> Estado actual: **MVP funcional** para compilación básica y empaquetado, con soporte inicial para flujo de plugins Minecraft (generación de `plugin.yml` y validación de JARs externos declarados en `kujav.toml`).
